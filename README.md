# Books
Api for CRUD books


## System Requirements ##
1. docker and docker compose
2. Git

## How to set up ##
1. Clone git reposotory.
   ```
   git clone https://github.com/sergio-ser/books.git
   ```
2. Build docker image.
    ```
    docker compose build
    ```
3. Up docker image.
    ```
    docker compose up
    ```

## Api documentation ##
1. Access the endpoint to see Api functional.
